# Chrome Web Store Test Instructions

1. Install the submitted extension package.
2. Open `https://chatgpt.com/`.
3. Open any conversation with ordinary text and, if available, a math formula.
4. Use the floating export controls to export:
   - one assistant response as DOCX;
   - one assistant response as PDF.
5. Confirm both files download locally and contain the selected response.
6. Open settings and confirm paper size, prompt inclusion, language, diagnostics, and copy target can be changed.
7. Optional WPS path:
   - select `WPS Office` as the copy target;
   - install the local WPS helper setup package if needed;
   - grant the optional Native Messaging permission;
   - recheck helper status;
   - copy a supported formula and paste into WPS Writer.

The default DOCX/PDF export and Microsoft Word copy paths do not require the WPS helper.

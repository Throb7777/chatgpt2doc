# Chrome Web Store Listing - English

## Name

ChatGPT2Doc

## Short Description

Export ChatGPT conversations locally to editable DOCX and searchable PDF.

## Detailed Description

Free, local export of ChatGPT conversations to documents you can keep, edit,
search, and share. No subscription is required.

ChatGPT2Doc adds small DOCX and PDF actions directly to ChatGPT. Export
one assistant response, the full conversation, assistant-only content, or a
set of messages you choose.

The exported document can preserve headings, paragraphs, lists, quotes, links,
citations, tables, code, images, Chinese and English text, and mathematical
expressions. Supported equations stay editable in Microsoft Word. PDF files
keep searchable text and embedded fonts. When a page element cannot be
converted safely, it remains visible through an explicit fallback instead of
being silently removed.

You can also copy supported formulas to Microsoft Word through the normal rich
clipboard. WPS Writer users may install the optional local Windows helper setup
package from the project release downloads for editable WPS equations. Chrome
requires this helper to be a separate user-approved Native Messaging component;
it is not required for DOCX, PDF, or Microsoft Word copy.

Privacy is simple: conversation parsing and document generation happen on your
device. There is no ChatGPT2Doc account, cloud conversion server,
subscription, analytics, or telemetry. The optional WPS helper also runs
locally and opens no network port.

Settings include English or Simplified Chinese, A4 or Letter paper, custom
file name, prompt inclusion, document theme, code style, optional diagnostics,
and Word/WPS copy target.

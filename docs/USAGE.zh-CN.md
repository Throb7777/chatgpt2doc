## 安装方法

### 通过 Chrome 应用商店安装（推荐）

应用审核通过并公开上架后，此处将添加直接安装链接。普通用户建议首选此方式，以便 Chrome 能够自动为您更新扩展程序。

源码与发行版下载地址：

- GitHub 仓库：[https://github.com/Throb7777/chatgpt2doc](https://github.com/Throb7777/chatgpt2doc)
- Releases 页面：[https://github.com/Throb7777/chatgpt2doc/releases](https://github.com/Throb7777/chatgpt2doc/releases)

### 本地构建与手动加载

1. 安装 Node.js 和 npm。
2. 在项目根目录下打开终端。
3. 运行构建命令：

   ```powershell
   npm ci
   npm run build:chrome
   ```

4. 打开 Chrome 浏览器并访问 `chrome://extensions/`。
5. 开启右上角的**“开发者模式”**开关。
6. 点击**“加载已解压的扩展程序”**。
7. 选择项目目录下的 `.output/chrome-mv3` 文件夹。
8. 打开或刷新 ChatGPT 网页端。

Microsoft Edge 用户：请运行 `npm run build:edge`，随后在 `edge://extensions/` 中加载 `.output/edge-mv3` 文件夹。

## 导出功能与操作

### 导出单条助手回复

将鼠标悬停在某条助手回复上，点击其旁边的 DOCX 或 PDF 操作图标，即可仅将该条回复单独导出。

### 导出完整对话

点击悬浮面板（胶囊）中的 DOCX 或 PDF 图标，对话消息将按页面中的显示顺序完整导出。

### 仅导出助手回复

1. 点击悬浮面板上的齿轮（设置）图标。
2. 关闭**“包含用户提示”**选项。
3. 照常通过悬浮面板导出完整对话。

此时，导出的文档将自动过滤掉您的用户提示词，仅按原顺序保留助手回复。

### 自定义导出指定消息

1. 在悬浮面板上点击**“选择消息”**。
2. 勾选您需要导出的用户提问或助手回复。
3. 点击屏幕底部操作栏中的 DOCX 或 PDF 按钮完成导出。

未勾选任何消息时无法进行导出。此外，若 ChatGPT 仍处于流式生成回复的“打字”状态，请耐心等待生成结束后再行导出。

## 导出进度与诊断提示

同一时间仅支持运行一个导出任务。在导出过程中，进度提示会显示当前处于“内容收集”、“文档生成”或“下载”阶段。您可以在文件正式开始下载前随时取消任务。文件生成完毕后，提示窗口会自动关闭。

默认状态下，成功导出仅显示“已完成”。如果您希望在遇到图片无法加载、公式暂不支持、内容收集不全或页面内容回退时查看详细的排版细节，可以在设置中开启**“显示导出诊断”**。

提示：文档中出现“回退”（Fallback）并不代表整个文件导出失败，仅表示该处特定的复杂内容无法完全以原本的复杂结构进行渲染。

## 输出细节与兼容性

- **Word 公式：**受支持的数学表达式将转换为原生的、可编辑的 Word 公式。
- **不支持的公式：**如果公式中包含暂不支持的特定写法，系统会自动通过图片或文本回退的形式进行渲染展示，绝不进行静默删除。
- **PDF 搜索：**导出的 PDF 文档内文本保持可搜索状态。
- **图片嵌入：**当浏览器能够成功读取并解码图片源地址时，图片会自动内嵌至文档中。
- **丢失的图片：**如果遇到无法正常嵌入的图片，扩展程序将尽量在原位保留图片原链接或显示一个可见的占位提示。

## 复制数学公式

### 复制公式至 Microsoft Word（无需辅助工具）

默认复制目标即为 Microsoft Word，无需安装任何辅助工具：

1. 在 ChatGPT 消息中选中需要的文本与数学公式。
2. 按下快捷键 `Ctrl+C`（Mac 上为 `Cmd+C`）。
3. 直接粘贴至 Microsoft Word 中。

在受支持的 Windows 版 Word 中，这些公式会自动转换为原生的、可编辑的 Word 公式。请注意，如果您在粘贴时选择“仅保留文本”，公式的结构格式将被自动去除。

### 复制可编辑公式至 WPS Writer（可选 Helper）

由于 WPS Office 使用不同的本机剪贴板格式，复制生成可编辑的 WPS 公式需要安装可选的 Windows 辅助工具（Helper）。Chrome 和 Edge 出于安全原因不允许应用商店扩展在用户电脑上静默安装或注册 Native Messaging 主机，因此该 Helper 必须作为一个由用户主动安装的本地组件存在。Release 下载中会提供可直接使用的 Helper 包；从源码构建只适合开发场景。

1. 从 [Releases](https://github.com/Throb7777/chatgpt2doc/releases) 下载 `chatgpt2doc-wps-helper-v1.0.0.zip` 并解压。
2. 在 ChatGPT2Doc 设置中或 `chrome://extensions/` 页面中找到当前扩展 ID。
3. 按照 [WPS Helper 说明](../native/wps-helper/README.md)，使用该扩展 ID 运行 Helper 的 `install.ps1`。
4. 打开 ChatGPT2Doc 设置。
5. 将“复制目标”更改为 **WPS Office**。
6. 当 Chrome 弹出提示时，允许可选的“本机消息传递”（Native Messaging）权限。
7. 点击“重新检查”，直至 Helper 状态显示为“可用”。
8. 在 ChatGPT 消息中选中内容并按下 `Ctrl+C`。
9. 直接粘贴至 WPS Writer 中。

标准的 DOCX 和 PDF 导出功能本身并不依赖此 Helper。

## 配置与设置说明

点击悬浮面板（胶囊）中的齿轮图标，即可个性化配置以下选项：

- **语言：**切换英文或简体中文。
- **文件名：**可自定义文件名，留空时默认使用“对话标题 + 时间戳”。
- **纸张：**选择 A4 或 Letter 纸张尺寸。
- **文档主题：**选择浅色（Light）或深色（Dark）主题样式。
- **代码样式：**设置代码块样式为“跟随文档”、“浅色”或“深色”。
- **包含用户提示：**决定导出整段对话时是否将用户的提问一并打包。
- **显示导出诊断：**开启或关闭详细的警告与回退诊断信息。
- **每条回复操作：**显示或隐藏 AI 助手回复旁的快捷导出按钮。
- **复制目标：**选择您的默认公式剪贴板目标（Microsoft Word 或 WPS Office）。
- **胶囊位置：**可任意拖动悬浮面板（胶囊），其位置将被浏览器自动记住。

点击**“重置设置”**将恢复出厂默认值。所有的个性化配置仅保存在您当前浏览器的本地扩展存储中。

## 常见问题与故障排查

- **页面上没有显示导出按钮：**安装或重新加载扩展程序后，请刷新 ChatGPT 网页。并确保在浏览器的扩展管理中，该扩展已获准在 `chatgpt.com` 上运行。
- **导出任务一直处于忙碌状态：**请先等待当前回复流式生成完毕。若依然卡住，请取消当前任务并重试一次。请避免同时发起多个导出任务。
- **公式显示为回退内容：**在设置中开启“显示导出诊断”以检查对应公式。若公式中使用了暂不支持的特殊语法，系统会使用回退文本或图示，以确保内容可见且不丢失。
- **导出的文档中图片丢失：**这通常是因为浏览器无法读取或解码该图片的源地址。扩展程序会尽量在原位置保留图片链接或显示一个可见的占位提示。
- **WPS Helper 提示不可用：**请确认设置中显示的扩展 ID，使用该 ID 重新安装 Helper，授予所需的权限后再点击“重新检查”。
- **手动加载时提示缺少 manifest 文件：**请确保已在终端中成功运行了 `npm run build:chrome`。在 Chrome 中加载时，应选择 `.output/chrome-mv3` 文件夹，而不是项目的根目录。

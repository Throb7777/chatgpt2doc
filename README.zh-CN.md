## 功能介绍

ChatGPT2Doc 直接在 ChatGPT 网页端界面中集成 DOCX 和 PDF 导出控制栏。

您可以轻松导出：

- 单条 AI 助手回复；
- 完整对话记录（按页面显示顺序）；
- 仅助手回复（自动过滤掉您的用户提示词）；
- 自定义勾选的多条消息。

导出时支持保留标题、列表、引用、链接、参考来源（Citations）、表格、代码块、图片、中英文和数学公式。

- **DOCX：**受支持的数学公式在 Word 中保持原生且可编辑的状态。
- **PDF：**生成的 PDF 文本支持搜索，并会自动内嵌所需字体。
- **本地处理：**所有文件均在您的本地设备上生成。本扩展无需注册账号，不使用任何外部转换服务器，且无订阅、无数据分析及遥测行为。

## 安装指南

### 通过 Chrome 应用商店安装（推荐）

商店审核通过并公开上架后，此处将添加直接安装链接。推荐普通用户使用此方式，以便浏览器后续自动更新。

源码与 Release 下载地址：

- GitHub 仓库：[https://github.com/Throb7777/chatgpt2doc](https://github.com/Throb7777/chatgpt2doc)
- 发行版下载：[https://github.com/Throb7777/chatgpt2doc/releases](https://github.com/Throb7777/chatgpt2doc/releases)

### 手动开发者模式安装

如果您希望通过源码手动加载当前版本：

1. 安装 Node.js 和 npm。
2. 在项目根目录下打开终端。
3. 运行以下构建命令：

   ```powershell
   npm ci
   npm run build:chrome
   ```

4. 打开 Chrome 浏览器并访问 `chrome://extensions/`。
5. 开启右上角的**“开发者模式”**开关。
6. 点击**“加载已解压的扩展程序”**按钮。
7. 选择项目目录下的 `.output/chrome-mv3` 文件夹。
8. 打开或刷新 ChatGPT 网页端。

Microsoft Edge 用户：请运行 `npm run build:edge`，随后在 `edge://extensions/` 中加载 `.output/edge-mv3` 文件夹。

## 如何导出文档

### 导出单条回复

将鼠标悬停在某条 AI 助手回复上，点击其旁边的 DOCX 或 PDF 图标即可单独导出该回复。

### 导出完整对话

点击悬浮面板（胶囊）中的 DOCX 或 PDF 图标，对话消息将按页面中的显示顺序完整导出。

### 仅导出 AI 助手回复

1. 点击悬浮面板上的**齿轮（设置）**图标。
2. 关闭**“包含用户提示”**选项。
3. 通过悬浮面板导出完整对话。

此时，导出的文档将自动过滤掉您的提问，仅按原顺序保留助手回复。

### 自定义导出指定消息

1. 在悬浮面板上点击**“选择消息”**。
2. 勾选您需要导出的用户消息或助手回复。
3. 点击屏幕底部操作栏中的 DOCX 或 PDF 按钮。
4. 完成后点击退出选择模式。

未勾选任何消息时无法导出。若 ChatGPT 仍在生成回复，请等待生成完毕后再进行导出。

## 导出进度与诊断说明

同一时间仅支持运行一个导出任务。在导出过程中，进度提示会实时显示当前处于“内容收集”、“文档生成”或“下载”阶段，您可以在文件开始下载前随时取消。文件生成完毕后，提示窗口会自动关闭。

默认状态下，成功导出仅显示“已完成”。如果您需要了解图片加载失败、公式暂不支持、内容收集不全或页面回退等详细技术细节，可以在设置中开启**“显示导出诊断”**。

提示：文档中出现“回退”（Fallback）并不代表整个文件导出失败，仅表示这部分特定内容无法完全以原本的复杂排版（如复杂的数学公式）进行渲染。

## 复制数学公式

### 复制至 Microsoft Word（无需辅助工具）

您无需安装任何辅助工具，即可将数学公式直接复制粘贴到 Word 中：

1. 在 ChatGPT 消息中直接框选需要的文本与数学公式。
2. 按下快捷键 `Ctrl+C`（Mac 上为 `Cmd+C`）。
3. 直接粘贴到 Word 文档中。

在受支持的 Windows 版 Word 中，受支持的公式会自动转换为原生的、可编辑的 Word 公式。注意：如果粘贴时选择“仅保留文本”，公式结构将被丢弃。

### 复制可编辑公式至 WPS Writer（可选）

由于 WPS Office 使用不同的本机剪贴板格式，复制生成可编辑的 WPS 公式需要安装可选的 Windows 辅助工具（Helper）。

操作步骤：

1. 按照 [WPS Helper 说明](native/wps-helper/README.md)，针对当前扩展 ID 构建并安装辅助工具。
2. 打开 ChatGPT2Doc 设置。
3. 将“复制目标”更改为 **WPS Office**。
4. 当 Chrome 弹出提示时，允许“本机消息传递”（Native Messaging）权限。
5. 点击“重新检查”，直至 Helper 状态显示为“可用”。
6. 在 ChatGPT 消息中选中内容并按下 `Ctrl+C`。
7. 直接粘贴至 WPS Writer 中。

在公式结构受支持的情况下，公式在 WPS 中将保持完全可编辑。如果 Helper 不可用，ChatGPT2Doc 会自动降级，安全地使用普通的 Word 兼容剪贴板格式。DOCX 和 PDF 导出功能本身并不依赖此 Helper。

## 配置与设置说明

点击悬浮面板（胶囊）中的齿轮图标，即可个性化配置以下选项：

- **语言：**切换英文或简体中文。
- **文件名：**自定义文件名，留空时默认使用“对话标题 + 时间戳”。
- **纸张：**选择 A4 或 Letter 尺寸。
- **文档主题：**选择浅色（Light）或深色（Dark）文档主题。
- **代码样式：**设置代码块样式为“跟随文档”、“浅色”或“深色”。
- **包含用户提示：**决定导出整段对话时是否包含用户的提问。
- **显示导出诊断：**开启或关闭详细的警告与回退诊断信息。
- **每条回复操作：**显示或隐藏助手回复旁的快捷导出按钮。
- **复制目标：**选择您的默认剪贴板公式兼容目标（Microsoft Word 或 WPS Office）。
- **胶囊位置：**可任意拖动悬浮面板（胶囊），其位置将被浏览器自动记住。

点击**“重置设置”**将恢复默认配置。所有的个性化设置仅保存在当前浏览器的本地扩展存储中。

## 常见问题与排错

- **页面中未显示导出按钮：**安装或手动加载扩展后，请刷新 ChatGPT 页面。确保在浏览器的扩展管理中，已允许该扩展在 `chatgpt.com` 域名上运行。
- **导出任务一直处于忙碌状态：**请先等待当前 ChatGPT 消息完全生成结束。若依然卡住，可以取消当前导出并重新尝试，避免同时发起多个导出任务。
- **公式显示为回退内容：**开启设置中的“显示导出诊断”以检查具体公式。如果公式中使用了某些暂不支持的特定排版语法，系统会自动以回退排版显示以确保内容不丢失。
- **导出的文档中图片丢失：**这通常是由于浏览器无法成功访问或解码该图片的源地址。扩展会尽最大可能在原位置保留图片链接或显示一个可见的占位符。
- **WPS Helper 提示不可用：**请确认设置中显示的扩展 ID，使用该 ID 重新安装 Helper，授予所需权限后再点击“重新检查”。
- **手动加载时提示缺少 manifest 文件：**请确保已在终端中成功运行了 `npm run build:chrome`。在 Chrome 中加载时，应选择 `.output/chrome-mv3` 文件夹，而不是项目的根目录。

## 隐私声明

您的数据隐私至关重要。所有对话内容仅在您主动进行导出或复制操作时，在本地浏览器中进行处理。ChatGPT2Doc 绝不会将您的聊天记录上传至任何服务器，也未集成任何追踪、分析或遥测服务。为了能在导出的文档中正常显示，对话中包含的远程图片可能会从其原始链接下载以完成嵌入。如需查看完整的隐私协议，请参阅 [PRIVACY.zh-CN.md](PRIVACY.zh-CN.md)。
